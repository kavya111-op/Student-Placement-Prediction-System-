# Student-Placement-Prediction-System-
This project aims to solve this real-world problem by using Machine Learning to predict whether a student is likely to be placed based on academic and skill-based factors.
# 🎓 Student Placement Predictor (AI/ML Project)

## 📌 Overview

This project predicts whether a student will be placed based on:

* CGPA
* Skills
* Internships

It uses a **Random Forest Machine Learning model** and provides an **interactive GUI inside Jupyter Notebook** using ipywidgets.

---

## 🚀 Features

✔ Interactive sliders (Jupyter GUI)
✔ Instant placement prediction
✔ Placement probability (%)
✔ Graph visualization
✔ Feature importance analysis

---

## 🧠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* ipywidgets

---

## ⚙️ How to Run

### Step 1: Install dependencies

```
pip install -r requirements.txt
```

### Step 2: Open Jupyter Notebook

```
jupyter notebook
```

### Step 3: Run the file

Open `app.ipynb` and click **Run All**

---

## 📊 Input Parameters

* CGPA (0–10)
* Skills (1–5)
* Internships (0–2)

---

## 📤 Output

* Placement Prediction (Placed / Not Placed)
* Placement Probability (%)
* Graphs:

  * Probability chart
  * Feature importance

---

## 🎯 Problem Statement

Students often lack clarity about their placement chances.
This project uses Machine Learning to provide data-driven predictions.

---

## 🧪 Model Used

**Random Forest Classifier**

Why?

* High accuracy
* Handles multiple inputs well
* Robust for small datasets

## 📉 Limitations

* Small dataset
* Basic features only
* Not industry-level accuracy
